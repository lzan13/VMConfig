VMConfig
========

保存自己平常常常用的一些配置，比如环境变量，shell 命令，alias


### 别名配置
`vm_alias.sh` 保存自己常用的别名配置，包括 adb、keytool、git 等相关命令

### 环境变量配置
`vm_env.sh` 保存环境变量配置，这个需要根据自己电脑目录进行配置

### 常用 gitignore
GitHub 已经写好了一份 .gitignore 供大家使用：
[常用 gitignore](https://github.com/github/gitignore)
